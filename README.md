## Technologies
- 100% Kotlin.
- Made Using Jetpack Compose.
- Following Clean Architecture approach.
- Following MVVM Architectural Design Pattern.

### Used in the app :
- Room DB
- Hilt
- Kotlin coroutines
- Kotlin Flows
- Jetpack Compose
